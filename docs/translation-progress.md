## 目标与范围

- 以 `book/chapter-01.html` 至 `book/chapter-22.html` 为英文源稿，整体重译 `zh/` 中的简体中文版。
- 译文以中国大陆读者为主要受众，优先保证准确、自然、具体、易懂；避免生硬直译和不必要的抽象表达，必要时按原意重组句子。
- 代码、变量、API、公式、引用、链接和 HTML 语义结构保持不变。
- 每章先完成重译，再由独立 agent 对照英文源文审查。审查重点为：语义是否准确、中文是否得体、是否存在遗漏或多译、术语是否一致。
- `front-matter.html`、`index.html` 和 `book-index.html` 不计入 22 章的逐章翻译进度；前置页、目录和书后索引在全书收尾时单独翻译与复核。

## 状态定义

- 翻译：`未开始` → `翻译中` → `已翻译` → `已修订`
- 审查：`未审查` → `审查中` → `有修改意见` → `通过`
- 自动检查：每章至少核对标题、正文、列表、图表说明和脚注覆盖；同时确认代码和公式未被改动。全书完成后运行全部中文质量与结构审计。
- “完成”仅指该章已经根据独立审查意见修订，且章节级自动检查通过。
